## â“ Troubleshooting (Kalau Ada Error)

### Error: "items.json not found"
**Masalah:** File `items.json` tidak ditemukan
**Solusi:** 
- Pastikan file `items.json` ada di folder yang sama dengan project
- Cek spelling nama file (harus persis `items.json`)

### Error: "npm command not found" atau "bun command not found"
**Masalah:** Node.js atau Bun belum terinstall
**Solusi:**
- Install Node.js dari https://nodejs.org/
- Atau install Bun dari https://bun.sh/

### Error: "Cannot find module" atau "Dependencies not installed"
**Masalah:** Dependencies belum di-install
**Solusi:**
- Jalankan `install.bat` atau ketik `npm install` di terminal

### File `ItemsEnum.h` tidak terbuat
**Masalah:** Ada error saat generate
**Solusi:**
- Pastikan file `items.json` valid (format JSON benar)
- Cek apakah ada pesan error di terminal
- Pastikan folder project memiliki izin write
